1.1 人工智能的定义与发展

定义：

  • 智能机器：能够在各种环境中自主地或交互地执行各种拟人任务的机器

  • 70年代：低谷；以自然语言理解、知识表示的研究为主

2.1 状态空间法

元素：

  • 状态：用于描述同类事物的不同实例而引入的一组最少的分量组成的有序的集合
  • 算符/操作符：使问题从一种状态变化为另一种状态的手段
  • 问题的状态空间：表示问题的全部可能状态及其关系的图

状态空间可以表示为三元组(S,F,G)21

  • S：初始状态集合
  • F：算符集合
  • G：目标状态集合

状态空间表示问题的步骤：

  • 定义状态的描述形式
  • 表示出所有状态，并确定初始状态集合和目标状态集合
  • 定义一组算符

问题的解：从问题的初始状态出发到目标状态所经过的所有算符构成的序列

2.2 产生式表示法

概述

产生式表示：基于规则的表示方法，主要应用于专家系统中

特点：

  • 清晰
  • 模块性
  • 自然性

产生式可表示的知识种类：

  • 事实性知识
  • 规则知识

产生式表示方法（可用元组表示）：

  • 确定性事实：if P then Q
  • 不确定性事实：if P then Q (置信度)

产生式系统的组成：一组产生式的相互配合，互相推理来得到结果的系统

  • 规则库
  • 综合数据库
  • 推理机

规则库

  • 规则库是用于描述某个领域内知识到产生式集合，是某个领域知识到存储器
  • 规则库包含将问题从初始状态转换成目标状态的转换规则
  • 规则库是系统的核心，知识到完整性、一致性、准确性和灵活性都对知识的性能和运行效率产生直接的影响

综合数据库

  • 综合数据库用于存放输入的事实，它们可从规则推理得到
  • 综合数据库是不断改变的

推理机

推理机的动作：

  • 匹配：将数据库中的知识和规则中的条件相比较
  • 冲突解决：有多种规则匹配数据库的同一条信息时，需要进行排序取舍，确定执行策略
  • 操作：修改数据库

推理机的推理方式

  • 正向推理
  • 反向推理
  • 双向推理

2.4 语义网络法

语义网络

语义网络是知识表示的一种结构化图解表示，它由节点和弧线或连接线表示

  • 节点：实体、概念或情况，每个节点可以有若干个属性（来区分不同对象）
  • 弧线：节点间的语义关系

语义网络可由三元组（节点1，弧，节点2）表示

也可以表示成谓词关系：P（节点1，节点2）；P表示弧的含义

语义网络表示知识到方法

步骤：

  • 确定问题中的所有对象以及各个对象的属性
  • 确定所讨论对象的关系
  • 整理同一类对象的共同属性，将它们抽象
  • 将各对象标注成节点，并用弧线表示它们之间的关系

事实知识的表示：

  • 可以用节点和有向边表示
  • 事实知识是有关领域内的概念、定义等事实性知识

规则性知识表示方法：

  • 弧线上的标注用“如果…那么…”表示

语义网络常见关系

  • ISA/AKO关系：ISA偏向于一种类和实例的关系、AKO偏向于子类和父类的关系
  • Part-of关系：用于表示部分和整体的关系，部分不一定具有整体的所有属性
  • IS关系：用于表示一个节点是另一个节点的属性
  • If-then关系：表示因果关系
  • Composed-of关系：表示组合关系，与Part-of有区别
  • Have关系
  • Located关系

语义网络的推理过程

  • 根据提出的待求解问题，构造一个局部网络或网络片段
  • 根据局部网络到知识库中去寻找语义网络，匹配并得到结果

2.5 框架表示法

框架

框架是一种所论对象属性的数据结构。该对象可以是一个事物、一个事件或者一个概念

  • 框架由若干个槽（slot）组成，用于描述对象某一方面的属性
  • 每个槽可以划分成若干个侧面（facet），用于描述相应属性的一个方面
  • 槽和侧面所具有的属性值分别称为槽值和属性值

框架表示法基本步骤

  • 分析待表达知识中的对象及其属性，对框架中的槽进行定义
  • 分析各对象之间的关系
  • 定义槽中的侧面和各侧面的属性值

框架法的推理过程

  • 将求解问题用一个框架表示出来
  • 将该框架与知识库中已有框架进行匹配
  • 合并冲突
  • 将待解决问题框架中需要求解的属性进行补充

三、确定性推理方法

概述

知识和推理共同构成智能

推理：

  • 确定性推理（经典逻辑推理）
  • 不确定性推理

3.1 推理的基本概念

推理的方式和分类

推理是依据已知事实和某种策略来得出结论的过程

推理方式及其分类：

  • 演绎推理：三段论
  • 归纳推理：完全归纳法（数学归纳法），不完全归纳法（抽样调查）
  • 默认推理：在知识不完全的情况下假设某些条件已经具备所进行的推理（有时不成立）
  • 确定性推理：推理时所用到的知识和证据都是确定的，推出的结论也是确定的
  • 不确定性推理：
  • 单调推理：推理所得出的结论越来越接近最终目标
  • 非单调推理：对真理的一种螺旋式认识，有时会否定之前的结论
  • 启发性推理：运用能加快推理、提高搜索效率的知识（启发式知识）进行推理
  • 非启发性推理

推理的方向

  • 正向推理：P53

    • 确定匹配方法
    • 按什么策略搜索知识库
    • 冲突消解策略
    • 简单、易实现，但目的性不强，效率低

  • 逆向推理

    • 目的性强，有利于向用户提供解释；但选择初始目标具有盲目性

  • 混合推理

  • 双向推理

冲突消解策略

已知事实与知识的匹配情况：

  • 一对一匹配
  • 不能匹配
  • 一对多或多对多的匹配

消解策略：

  • 按针对性排序
  • 按事实新鲜性（时间）排序
  • 按匹配度排序
  • …

3.2 自然演绎推理

从已知为真的事实运用经典逻辑推理规则推出结论

  • P规则：给出的结论
  • T规则：综合推导出的结论
  • 假言推理：$P, \ P \to Q \Rightarrow Q$
  • 拒取式推理：$P \to Q, \lnot Q \Rightarrow \lnot P$

3.3 谓词公式化为子句集的方法

子句是谓词公式及其否定的析取式

  • 不包含任何谓词公式的子句为空子句
  • 空子句永假

子句集是子句的集合，子句间的关系是合取

谓词公式化为子句集：

  • 消去公式中的等价和蕴含箭头
  • 展开带否定符号的公式
  • 量词代还、变量标准化
  • 消去存在量词：无全称管辖的存在量词常量化，有全称管辖的量词skolem化，即$P(x,y) \Rightarrow P(x,f(x))$
  • 化为前束范式：量词提前
  • 主公式化为子句和取形式
  • 略去全称量词
  • 消去和取词，形成子句集
  • 子句变量标准化：即替换变量使得不同子句中的变量不同，因为子句之间是和取关系

3.4 鲁滨逊归结原理

鲁滨逊归结原理（消解原理）用于谓词公式的不可满足性分析（充分条件）

思想：子句集中子句之间是合区关系，只要有一个子句不可满足，则子句集就不可满足

  • 检查子句集S中是否包含空子句，若有则S不可满足
  • 从子句集中寻找合适的子句进行归结，若结果得到空子句，则S不可满足
  • 若没有归结出空子句，则不能下任何结论

归结过程：

  • C1和C2是两个子句
  • 取出两个子句中互补的文字（原子公式）并消去
  • 余下部分析取，得到C12
  • 此时，C1和C2为C12的亲本子句

在归结过程中消去带变元的谓词互补时，需要用到置换与合一的概念：https://blog.csdn.net/tianranhe/article/details/8287534

  • 合一置换之后，变元相同，直接消去即可

定理：

  • 归结式C12是亲本子句C1和C2的逻辑结论。
  • 使用C12替换C1和C2，得到归结后的子句集S1的不可满足性和原子句集S等价

3.5 归结反演

步骤：

  • 将前提化为谓词公式F
  • 将待证明结论表示为谓词公式Q，并得到否定not Q
  • 将谓词公式F and not Q化为子句集S
  • 对S进行归结，若出现空子句，则停止归结，此时Q为真

3.6 归结原理的应用

问题求解：

  • 将前提$P$和问题$\exist x Q(x)$（要求的是x）用谓词表示出来
  • 将问题取反并和$Answer$析取，即$\lnot \exist x Q(x) \land Answer(x)$
  • 将上述子句化为子句集，并进行归结
  • 若得到归结式$Answer$，则答案就在$Answer$中

四、不确定性推理方法

概述

现实世界客观存在随机性和模糊性，需要进行不确定性的表示和推理

基于概率论的不确定性的推理方法

不确定性推理：

从不确定的初始证据出发，通过运用不确性的知识，最终推理出一定程度的不确定性但是合理或近乎合理的结论的思维过程

不确定性的表示与度量：

  • 知识到不确定性的表示：一般由专家系统中的专家给出；是一种数值形式的静态强度
  • 证据的不确定性的表示：用户在求解问题时提供的初始证据；是一种动态强度
  • 不确定性的量度：确定一个客观的不确定性的量度

要点：

  • 不确定性的匹配算法和阈值
  • 组合证据的不确定性算法
  • 不确定性传递算法
  • 结论的不确定性合成

4.2 可信度方法

可信度：根据经验对一个事物或现象为真的相信程度，其具有较大的主观性和经验性

C-F模型

知识的不确定性表示：IF E THEN H (CF(H,E))

  • C-F：可信度因子，反映前提条件与结论的联系强度（主观参数）
  • CF也可以理解成：E为真时对H的影响程度
  • CF$\in$[-1,1]: 可信度从假到真；0为不相关

证据不确定性的表示：CF(E)

  • 此时CF是一个动态的强度，表示E为真的可能性

组合证据的不确定性：

  • 证据和取：CF(E) = min{CF(E1),CF(E2)…}
  • 证据析取：CF(E) = max{CF(E1),CF(E2)…}

不确定性的传递算法：CF(H)=CF(H,E)*max{0,CF(E)}

不确定性的合成算法：P82

算证据的可信度：

  • 算出每一条知识的结论的可信度
  • 合成结论的可信度得到结果的可信度

4.3 证据理论

概率分配函数

设D是样本x的样本空间；则在证据理论中，D的任何一个子集A都可以对应x的一个命题“x的值在A中”

概率分配函数M定义如下：
$$M: 2^{|D|} \to [0,1] 且 \
M(\empty)=0\
\sum_{A\in D}M(A)=1
$$

概率分配函数的正交和（概率组合问题）：P85-86

基于证据理论的不确定性推理

步骤：

  • 建立问题的样本空间
  • 由经验给出各概率分配函数（先组合证据计算可信度，再组合概率）
  • 计算结论的信任函数
  • 得出结论

遗传变异理论：繁殖、变异、竞争、选择

遗传理论

遗传是指父代利用遗传基因将自身的基因信息遗传给下一代，使子代能够继承父代的特征或性状的生命现象

概念：

  • 适者生存：目标值比较大的解被选择的可能性较大
  • 个体：解
  • 染色体：解的编码
  • 基因：解的编码中的每一分量，即染色体中的每一个位
  • 适应性：适应性函数，评估某个个体对当前环境的适应程度
  • 群体：根据适应度值选择的一组解；种群可以为初始情况下的解的集合
  • 婚配：交叉选择两个染色体进行交叉产生一组新的染色体的过程
  • 变异：编码的某一分量发生变化的过程

遗传算法

特点：通用、鲁棒、可能得到次优解（满意解），可以用于解决高度复杂的非线性问题

要素：

  • 参数编码
  • 初始种群设定
  • 适应性函数的设计
  • 选择操作
  • 交叉操作

流程：

  1. 编码、生成初始种群
  2. 计算种群个体对适应度并进行评价
  3. 判断是否满足终止条件，满足则终止
  4. 进行选择、交叉、变异，转2.

与普通优化（迭代）算法的区别：

  • 初始状态从一组点开始而非一个点开始
  • 易于形成通用算法程序
  • 容错能力强
  • 利用概率转移规则，算法是非确定性的

遗传编码

遗传编码：二进制编码、实数编码

二进制编码：格雷编码

  • 原编码中每一位和其前一位异或（最高位不变）

欺骗问题

选择操作

根据选择概率按照某种策略从当前种群中挑选出一定数目的个体，使它们能够有更多的机会被遗传到下一代中

  • 适者生存
  • 保持多样性：避免快速收敛到局部最优解

蒙特卡洛算法、排序算法、轮盘赌选择、锦标赛选择（随机选k个个体取其大加入集合，直到数量满足）

最佳个体保存：最佳个体不进行交叉直接复制到下一代

交叉操作

概述：

  • 单点（多点）交叉：将交叉点部分结构进行互换
  • 均匀交叉：产生一个随机掩码，1的部分交换，0的部分不交换
  • 交叉之后可能子代不符合要求，需要进行修正

交叉概率的选择：

  • 概率过高：破坏性较大
  • 概率过低：父代子代差异过小
  • 适合的概率：0.25-1

变异

对选中个体的染色体中点某些基因进行变动，以形成新的个体

  • 它可以增强种群的多样性
  • 二进制变异：随机产生一个或多个变异位
  • 实值变异：将某一个位置替换成一个符合范围的随机数
  • 基于位置的变异：随机产生两个变异位置，将位置上的值进行交换…

变异是辅助性的，其概率不应太大：0.001左右

感知机

感知机是神经网络和SVM的基础

  • 二分类的线性分类模型

输入： 特征向量

输出： 实例的类别

模型：
$$
f(x) = signal(wx_i+b)
$$

学习策略：损失函数设计（误分类点到超平面距离之和），需要损失函数最小化
$$
\sum_i (-y_i(wx_i+b) if y_i(wx_i+b)<0 else 0)
$$

损失函数最小化策略：梯度下降算法

  • 选择一个分类错误的点：误分类点梯度下降
  • 随机选择点：随机梯度下降算法（速度快）
  • 每个样本都进行误差累加之后更新权重：普通梯度下降算法（速度慢）
  • 选部分样本进行梯度下降：批量梯度下降算法
    $$
    w = w + \eta y_ix_i \
    b = b + \eta y_i
    $$

感知机扩展：线性回归
$$
L(w,b) = \sum_{n=1}^N(y^n - (wx+b)^2)
$$

支持向量机

相比于感知机，支持向量机努力提高模型的泛华能力

基础模型：二分类，同感知机
$$
f(x) = signal(wx_i+b)
$$

目标：在特征空间上找出间隔最大的超平面

间隔：
$$
min_{w,b} \frac12 ||w||^2 \
s.t. y_i(wx_i+b) -1 \le 0
$$
